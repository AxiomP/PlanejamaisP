# Planeja+ MVP

Micro SaaS educacional com IA para professores gerarem conteúdo pedagógico alinhado à BNCC.

## 🚀 Stack Tecnológica

- **Frontend:** Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** Supabase (PostgreSQL)
- **Autenticação:** Supabase Auth
- **IA:** Google Gemini Pro
- **Pagamentos:** Mercado Pago
- **Deploy:** Vercel

## 🎯 Funcionalidades MVP (10 Ferramentas)

1. ✅ Plano de Aula
2. ✅ Prova/Avaliação
3. ✅ Lista de Exercícios
4. ✅ Sequência Didática
5. ✅ Texto de Apoio/Resumo
6. ✅ Ideias de Atividades
7. ✅ PEI/PDI (Educação Especial)
8. ✅ Reescritor/Nivelador de Texto
9. ✅ Projeto Educacional
10. ✅ Chatbot BNCC

## 📦 Instalação

```bash
# Clonar repositório
git clone <repository-url>
cd planejamais-mvp

# Instalar dependências
npm install

# Configurar variáveis de ambiente
cp .env.example .env.local
# Editar .env.local com suas credenciais

# Executar em desenvolvimento
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## 🗄️ Configuração do Supabase

1. Crie um projeto no [Supabase](https://supabase.com)
2. Execute as migrations em `/supabase/migrations`
3. Configure as variáveis de ambiente
4. Execute os seeds de dados iniciais
